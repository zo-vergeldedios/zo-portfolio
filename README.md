# ZoVDD Portfolio

Live Site: https://zovdd.dev

---

## Overview

This is my personal portfolio website built to showcase my projects, skills, and development journey as a web developer and designer.

The goal of this site is simple:
Build something clean, fast, and expressive that reflects how I think as a developer and designer.

---

## Tech Stack

- React (Frontend framework)
- Tailwind CSS (Styling)
- JavaScript (Core logic)
- Claude AI (Assisted in ideation, structure, and refinement of components and content)

---

## Features

- Responsive design (mobile-first)
- Clean UI/UX focused layout
- Project showcase section
- About me / developer story
- Contact section for opportunities
